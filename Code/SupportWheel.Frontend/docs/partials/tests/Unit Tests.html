Unit Tests for the Front-end

Karma

To run the tests for this solution you need to have KARMA runner installed in your system. It is recommended to install also the karma-cli as global, that way you can run it anywhere. To install:

    npm install -g karma-cli

To run the tests just a single time, type:

grunt karma:unit

or if you want it to be running all time, type: 

 karma start

HTML Report

The tests are part of documentation, as so when the docs are generated a version is created in HTML format. To check this version, follow the link below:

To update this documentation with the unit tests run the task below

grunt karma:unitHtml
